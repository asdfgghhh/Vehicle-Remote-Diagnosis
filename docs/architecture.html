车辆远程诊断系统（VRD）· 系统架构图

基于 Spring Cloud 微服务 + Vue3 前后端分离架构。
管理端自上而下经网关访问微服务；车端自下而上经 service-access 统一接入（MQTT/HTTP）。
Nacos 负责服务注册发现与配置管理，不参与业务数据主链路。
